# Conference ROI Framework

This repository outlines a structured approach I use to evaluate third-party conferences and large-scale brand events.

## Objectives
- Drive qualified lead generation
- Increase brand visibility in priority markets
- Support Sales with seller-ready follow-up assets
- Measure ROI beyond vanity metrics

## Pre-Event Strategy
- Define event tier (Tier 1, 2, 3 based on audience value)
- Align with Sales on ICP (Ideal Customer Profile)
- Set measurable targets:
  - Qualified leads
  - Meetings booked
  - Pipeline influenced
  - Sponsorship visibility metrics

## Onsite Execution
- Structured lead capture process
- Clear tagging in CRM (region, segment, priority)
- Executive engagement tracking
- Side-event attendance tracking

## Post-Event Measurement
- Leads by quality tier
- Cost per qualified lead
- Pipeline generated within 30–60 days
- Lessons learned & playbook refinement

This framework ensures conferences are treated as commercial growth levers rather than standalone marketing moments.
